# 🌱 Crop Recommendation and Fertilizer Prediction System

## 📌 Project Overview
The Crop Recommendation and Fertilizer Prediction System is a machine learning-based web application developed to help farmers choose suitable crops and fertilizers based on soil and environmental conditions.

This project predicts the best crop to cultivate using parameters such as:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH value
- Rainfall

The system also recommends suitable fertilizers to improve crop productivity and soil health.
